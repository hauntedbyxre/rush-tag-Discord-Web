Project Cheetah Discord Widget Website
======================================

Overview
--------
This repository contains a custom HTML website that embeds the Project Cheetah Discord server widget
and adds extra features for announcements, events, rules, and roles. The website is designed to
provide members and visitors with a clean and interactive view of the server.

Features
--------
- Official Discord server widget embedded
- Announcements/Chat panel with add/remove functionality (Server Sided SOON)
- Events panel with upcoming events
- Rules section for server guidelines
- Roles display panel
- Fallback button for browsers that block the widget

Setup & Usage
-------------
1. Download or clone the repository.
2. Open `discord-widget.html` in a web browser to test locally.
3. To host online:
   - GitHub Pages: Upload the repository and enable Pages in the repo settings.
   - Netlify or Vercel: Drag-and-drop or import the repo for instant hosting.
