Project Cheetah Discord Widget Website
======================================

Overview
--------
This repository contains a custom HTML website that embeds the Project Cheetah Discord server widget
and adds extra features for announcements, events, rules, and roles. The website is designed to
provide members and visitors with a clean and interactive view of the server.

Features
--------
- Official Discord server widget embedded
- Announcements/Chat panel with add/remove functionality (Server Sided SOON)
- Events panel with upcoming events
- Rules section for server guidelines
- Roles display panel
- Fallback button for browsers that block the widget

Setup & Usage
-------------
1. Download or clone the repository.
2. Open `discord-widget.html` in a web browser to test locally.
3. To host online:
   - GitHub Pages: Upload the repository and enable Pages in the repo settings.
   - Netlify or Vercel: Drag-and-drop or import the repo for instant hosting.
4. The HTML file is fully editable — you can customize announcements, events, roles, and styles.

HOW TO DOWNLOAD ON MOBILE
-------------------------
# IF YOU ARE ON MOBILE DO THIS TO OPEN THE HTML

On Android
    1.    Download the HTML file from a browser or email.
    2.    Open it with a file manager (like Files by Google).
    3.    Open with a browser:
    •    Tap the file → choose Chrome, Firefox, or Edge to open it.
    •    Some file managers allow you to “Open with” directly.
    4.    Optional: Install a code editor app (like AIDE, QuickEdit, or Dcoder) if you want to edit the HTML.

⸻

On iOS (iPhone/iPad)
    1.    Download the HTML file from Safari or email.
    2.    Save to Files app.
    3.    Open with a browser:
    •    Long-press the file → choose Share → Open in Safari (or another browser app like Chrome).
